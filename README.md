# DRP03 – Projeto Integrador em Computação I

Projeto desenvolvido para o curso de Ciência de Dados da UNIVESP.

## Sobre o Projeto

O objetivo deste projeto é desenvolver um sistema web de gestão para Consultoras Independentes Mary Kay, validado por uma Diretora de Vendas.

A aplicação permitirá o gerenciamento de:

- vendas
- estoque
- clientes

Além disso, o sistema incluirá dashboards de acompanhamento, aplicando conceitos de análise de dados.

## Objetivo Acadêmico

Desenvolver um software utilizando framework web, aplicando na prática:

- levantamento de requisitos
- desenvolvimento web
- HTML e CSS
- banco de dados
- controle de versão

## Funcionalidades Principais

- CRUD de vendas
- CRUD de estoque
- registro de clientes
- dashboard de acompanhamento de desempenho

## Tecnologias

Ainda em definição pelo grupo.

Possíveis tecnologias:

- Python
- Flask / FastAPI / Streamlit
- HTML / CSS
- Banco de dados relacional
- Git e GitHub
